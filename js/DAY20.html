JavaScript Dialogs

In JavaScript, you can create dialog boxes or popups to interact with the users.

There are three types of dialog boxes: alert, confirm and prompt boxes.

The dialog boxes stop the code execution when it is displayed and the code execution only resumes after the boxes are dismissed.

Creating Alert Dialog Boxes

Alert Boxes enable to display a short message to the user. It includes OK button.

Syntax: alert(message)

Creating Confirm Dialog Boxes

Confirm Boxes allows user to confirm or cancel an action. It includes OK and Cancel buttons which returns Boolean values True and False respectively.

Syntax: confirm(confirmation_message)

Creating Prompt Dialog Boxes

Prompt Boxes prompts the user to enter some information. It includes a text box and an OK and a Cancel button.

Syntax: prompt(prompt_message)

JavaScript DOM

DOM or Document Object Model is a platform or language independent model to represent the HTML documents. It defines the logical structure of the documents and the way in which they can be assessed and manipulated by an application program.

In the DOM, all parts of the document i.e., elements, attributes, text,etc. are organized in a hierarchical tree-like structure. These individual parts of the document are known as nodes.

JavaScript DOM Selectors

JavaScript provides different selectors to select the elements on a webpage and manipulate them.

Selecting Elements by ID

We can select an element based on its unique ID with the getElementById() method.

HTML: <p id="para"> This is a paragraph. </p>

Selection: document.getElementById("para")

Selecting Elements by Class Name

We can select all elements having specific class names with the getElementsByClassName() method.

HTML: <p class="para"> This is a paragraph. </p>

Selection: document.getElementsByClassName("para")

Selecting Elements by Tag Name

We can select the elements by tag name with the getElementsByTagName() method.

HTML: <p> This is a paragraph. </p>

Selection: document.getElementsByTagName("p")

Selecting Elements with CSS Selectors

We can also select elements that matches the specified selector with the querySelector() and querySelectorAll() methods.

HTML: <p class="para"> This is a paragraph. </p>

Selection: document.querySelector(".para")

If you have more than one elements with same classname and select it, use document.querySelectorAll('.para')

JavaScript DOM Styling

We can also apply style on HTML elements by selecting them dynamically using JavaScript.

HTML: <p class="para"> This is a paragraph. </p>

Script: document.querySelector(".para").style.color="red"

Since CSS has properties like font-size, border-left-style, etc. uses hyphen in their names and that hyphen (-) is reserved as minus in JavaScript, the properties with the hyphen symbols are written in camel case format like fontSize, borderLeftStyle, etc.

Getting Style Information

To get all the values of all CSS properties, we can use the window.getComputedStyle() method and to get the value of specific property, we can use getPropertyValue() method.

Example: <p id="para"> This is a para </p>

Script: var styles = window.getComputedStyle(elem); styles.getPropertyValue("color")

Adding CSS classes to Elements

We can also get or set CSS classes to the HTML elements using the className property.

Example: <div id="newDiv" class="newClass"></div>

Script: var div = document.getElementById("newDiv"); div.className = "changedClass"; div.className += " anotherChangedClass";

Manipulating CSS classes using classList

We can use the classList property to get, set or remove classes from an element.

Example: <div id="newDiv" class="newClass"></div>

Script: var div = document.getElementById("newDiv"); div.classList.add("changedDiv"); div.classList.remove("newClass"); div.classList.toggle("changedDiv");

We can also use the contains method to check if the class is present in the element or not.

Example: div.classList.contains("newClass");

JavaScript DOM Get/Set Attributes

JavaScript provides several methods for adding, removing or changing an HTML element's attribute.

We can use getAttribute() to get the current value of an attribute on the element. If the specified attribute doesnot exist, it will return null.

Example: <div id="newDiv" class="newClass"></div>

Script: var div = document.getElementById("newDiv"); var divID = div.getAttribute("id");

Similarly, we can use setAttribute() to set or update the current value of an attribute on the element.

Example: <div id="newDiv" class="newClass"></div>

Script: var div = document.getElementById("newDiv"); var divID = div.getAttribute("id"); div.setAttribute("title", "This is a new Div");

We can also use removeAttribute() to remove the attribute on the element.

Example: <div id="newDiv" class="newClass"></div>

Script: var div = document.getElementById("newDiv"); div.removeAttribute("class")

JavaScript DOM Manipulation

We can also add or remove elements dynamically using JavaScript.

Adding New Elements to DOM

We can use document.createElement() method to create a new element.

Example: var newDiv = document.createElement("div");

To add the element to the DOM, we use appendChild() method

Example: document.body.appendChild(newDiv);

Getting or Setting HTML Contents to DOM

We can get or set the contents of the HTML elements using the innerHTML property.

HTML: <div id="para"> This is a para </div>

Script: var contents = document.getElementById("para").innerHTML; contents.innerHTML = "This is a new para text";

Removing the existing elements from DOM

To remove the elements/node from the DOM, we use the removeChild() method.

HTML: <div id="main"><div id="node1"></div><div id="node2"></div></div>

Script: var main = document.getElementById('main'); var node2 = document.getElementById('node2'); main.removeChild(node2);

JavaScript DOM Navigation

Sometimes, we need to access a child or parent elements apart from selecting individual elements on a web page.

Accessing the Child Nodes

We can use firstChild and lastChild properties of the DOM node to access the first and last direct child node of a node, respectively. If no child elements are found, it returns null.

Similarly we can use firstElementChild and lastElementChild to return only the first and last element node by skipping any whitespace text nodes, respectively.

Accessing the Parent Nodes

We can use parentNode property of the DOM node to access the parent node. The parentNode will always return null for document node as it doesn't have a parent.

Accessing the Sibling Nodes

We can use the previousSibling and nextSibling properties to access the previous and next node in the DOM tree, respectively.

Similarly, we can also use previousElementSibling and nextElementSibling to get the previous and next sibling element skipping any whitespace text nodes.
