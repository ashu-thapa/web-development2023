JavaScript Window

In JavaScript, the window object represents a window containing a DOM document.

Calculating the Width and Height of the Window

The window object provides the innerWidth and innerHeight property to find out the width and the height of the browser window viewport (in pixels) including the horizontal and vertical scrollbar, if rendered.

Syntax: window.innerWidth, window.innerHeight

Similarly, we can use clientWidth and clientHeight to find out the width and height of the window excluding the scrollbars.

Syntax: document.documentElement.clientWidth, document.documentElement.clientHeight

JavaScript Location

The window.location is a reference to a Location object. It represents the current URL of the document being displayed in the window.

Getting the current page URL

We can use the window.location.href property ot get the entire URL of the current page.

Syntax: window.location.href

Getting different parts of URL

We can also get other properties of the location object as protocol, hostname, port, pathname, search, etc.

Examples:

<script>
console.log("Different parts of URL:");
console.log("Protocol: "+window.location.protocol);
console.log("Host: "+window.location.host);
console.log("Hostname: "+window.location.hostname);
console.log("Port: "+window.location.port);
console.log("pathname: "+window.location.pathname);
console.log("Search: "+window.location.search);
console.log("Hash: "+window.location.hash);
</script>

Loading New Documents

We can use the assign() method or set the href of the location object to load another resource from a URL provided as parameter.

Syntax: window.location.assign(newURL); or window.location.href = newURL;

Alternatively, we can also use the replace() to replace the contents of another resource. However, using replace() doesn't create an entry in the browser's history.

Dynamically reloading the page

We can use the reload() method of the page to reload the current page dynamically.

The method takes two optional parameters true or false. If true is specified, the browser will be forced to reload the page from the server and if false, the browser may reload the page from its cache.

Syntax: window.location.reload(true|false)

JavaScript History

The history property of the Window object contains the session history, a list of all the pages visited in the current window.

Getting the number of Pages Visited

The window.history.length property can be used to get the number of pages in the session history of the browser for the current window.

Syntax: window.history.length or history.length

Going Back to the Previous Page

The window.history.back() method can be used to go back in the session history of the browser for the current window.

Syntax: window.history.back() or history.back()

Going Forward to the Previous Page

The window.history.forward() method can be used to go forward in the session history of the browser for the current window.

Syntax: window.history.forward() or history.forward()

Going to the specific Page

The window.history.go() method can be used to go to the specific page in the session history of the browser for the current window.

This method takes an integer as a parameter. A positive integer moves forward while a negative integer moves backward in the history.

Syntax: window.history.go() or history.go()

JavaScript Navigator

The navigator property of a window is a reference to a read-only Navigator Object that provides information about the user's browser.

Detect Whether the Browser is Online or Offline

We can use the navigator.onLine property to detect wheter the browser is online or offline.

This property returns true if online and false if offline.

Syntax: navigator.onLine

Getting Browser Information

The Navigator object has five main properties: appName, appVersion, appCodeName, userAgent, platform

Syntax: navigator.appName, navigator.appVersion, etc.

Check if Cookies are Enabled or Not

We can use navigator.cookieEnabled to check whether cookies are enabled in the user's browser or not.

This property returns true if cookies enabled else returns false.

Syntax: navigator.cookieEnabled

Check if Java are Enabled or Not

We can use navigator.javaEnabled() method to check whether cookies are enabled in the user's browser or not.

This method returns true if java is enabled else returns false.

Syntax: navigator.javaEnabled()

JavaScript Timers

A timer is a function that enables us to execute a function at a particular time. There are two types of timer functions in JavaScript: setTimeout() & setInterval()

setTimeout()

The setTimeout() function is used to execute a function or specified piece of code just once after a certain period of time.

Syntax: setTimeout(function, milliseconds)

setInterval()

The setInteval() function is used to execute a function or specified piece of code repeatedly at fixed time intervals.

Syntax: setInterval(function, milliseconds)

clearTimeout()

The clearTimeout() method can be used to clear the timeout and takes one parameter, an ID associated with setTimeout() function.

Syntax: clearTimeout(timeoutID);

clearInterval()

The clearInterval() method can be used to clear the timeout and takes one parameter, an ID associated with setInterval() function.

Syntax: clearInterval(intervalID);

JavaScript Date/Time

JavaScript provides a built-in Date object that allows accessing user's local date and time through the browser. It also provides several methods for managing, manipulating, and formatting dates and times.

Create a Date object

Date() constructor is used to create a Date object.

Syntax: var d = new Date();

The Date() constructor accepts parameters separated by commas: year, month, day, hours, minutes, seconds, milliseconds

Syntax: var d = new Date(year, month, day, hours, minutes, seconds, milliseconds);

Date can also be created by passing dateString as parameter in the Date() constructor.

Syntax: var d = new Date(dateString);

Date can also be created by passing milliseconds as parameter in the Date() constructor.

Syntax: var d = new Date(milliseconds);

Get the current date and time

To get the current date and time, we can simply create a new Date object without passing any parameters.

Syntax: var now = new Date();

Creating the Date and Time Strings

Several methods such as toDateString(), toTimeString() and others can be used to generate date strings in different formats.

List of Date/Time Strings:

  • d.toDateString()
  • d.toLocaleDateString()
  • d.toISOString()
  • d.toUTCString()
  • d.toString()
  • d.toTimeString()
  • d.toLocaleTimeString()

Getting Specific Date and Time Components

We can also extract specific components from the Date object like month, day, hours, minutes, etc.

Getting the Year, Month and Date

Syntax: var d = new Date(); d.getDate(); d.getDay(); d.getMonth(); d.getFullYear();

Getting the Time, Hours, Minutes, Seconds and Milliseconds

Syntax: var d = new Date(); d.getTime(); d.getHours(); d.getMinutes(); d.getSeconds(); d.getMilliseconds();

Setting Year, Month, Date

The Date object provides setFullYear(), setMonth() and setDate() methods to set the year, month and date components of the Date Object respectively.

Syntax: var d = new Date(); d.setFullYear(); d.setMonth(); d.setDate();

Setting Hours, Minutes, Seconds and Milliseconds

The Date object provides setHours(), setMinutes(), setSeconds() and setMilliseconds() methods to set the hours, minutes, seconds and milliseconds components of the Date Object respectively.

Syntax: var d = new Date(); d.setHours(); d.setMinutes(); d.setSeconds(); d.setMilliseconds();
