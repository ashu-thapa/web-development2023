JavaScript Math Operations

JavaScript provides a Math object that provides a number of userful properties and methods for performing mathematical operations.

Math.PI Property

The Math.PI property represents the value of PI constant, which is approx 3.14159.

Syntax: Math.PI

Getting the Absolute Value

The Math.abs() method is used to calculate the absolute (positive) value of a number.

Syntax: Math.abs(number)

Generating a Random Number

The Math.random() method is used to generate a floating-point random number.

Syntax: Math.random() or Math.random(max)

Calculating the Square Root of a Number

The Math.sqrt() method is used to calculate the square root of a number.

Syntax: Math.sqrt(x)

Rounding up the Number

The Math.ceil() method rounds a number up to the next highest integer. Ex: 3.5 is rounded up to 4.

Syntax: Math.ceil(x)

Rounding down the Number

The Math.floor() method rounds a number down to the next lowest integer. Ex: 3.5 is rounded down to 3.

Syntax: Math.floor(x)

Rounding the Number

The Math.round() method rounds a number to the nearest integer in such a way that if the decimal part is .5 or greater, number is rounded up, else rounded down. Ex: 3.5 is rounded to 4, 2.3 is rounded to 2.

Syntax: Math.round(x)

Find the largest and smallest numbers

The Math.max() and Math.min() methods is used to find which number is the largest or smallest in a group of numbers, respectively.

Syntax: Math.max(numbers); Math.min(numbers);

To find the maximum or minimum value in an array, we use the apply() method.

Syntax: Math.max.apply(null, array); Math.min.apply(null, array);

Raising Numbers to a Power

The Math.pow() method is used to raise a number to a specified power.

Syntax: Math.pow(number, power)

JavaScript Type Conversions

Most of the time, JavaScript automatically converts values from one data type to another when used in expression.

But, there can be situations when we need to manually convert a value from one data type to another.

Converting Values to Numbers

We can use Number() method to convert strings to numbers.

Syntax: Number(x); x can be strings, boolean, null, undefined or other expressions

Converting Values to Strings

We can use String() method to convert a value to a string.

Syntax: String(x); x can be numbers, boolean, null, undefined or other expressions

Type Conversions using Operators

Certain JavaScript operators like + and - can also be used to perform type conversions.

+ operator in front of a string converts it into a number. Example: var x = "20"; var y = +x; Here, x is converted to a Number.

Similarly, if we use the + operator to add a number to an empty string, the number is converted to a string.

JavaScript Event Listeners

The event listeners are like event handlers, however in event listeners, we can assign as many event listeners to a particular event on particular element.

Syntax: target.addEventListener(event, function, useCapture)

We can also use removeEventListener(event, function) to remove an event listener that was previously attached with the addEventListener().

JavaScript Event Propagation

Event propagation is the mechanism that defines how the events propagate or travel through the DOM tree to arrive at its target and what happens to it afterwards.

Phases of Propagation

  • Capturing
  • Bubbling
  • Targeting

Accessing the Target Element

The target element is the DOM node that has generated the event. The event.target property can be used to get the target using JavaScript.

Stop Propagation

We can use stopPropagation() and stopImmediatePropagation() methods to stop event propagation in the middle.

Prevent Default

Some events have a default action associated with them like submit button submits the form and reloads the page or an anchor tag redirects to the link's target on click.

Sometimes, there may be need to prevent such default actions using preventDefault() method of the event object.

JavaScript JSON Parsing

JSON stands for JavaScript Object Notation. It is an extremely lightweight data-interchange format for data exchange between server and client, easy and quick to parse and generate.

JSON is based on two basic structures:

  • Objects: key/value pairs
  • Arrays: list of values

Parsing JSON Data in JavaScript

In JavaScript, you can easily parse JSON data received from the web server using the JSON.parse() method.

Encoding Data as JSON in JavaScript

Sometimes, JavaScript object or value from the code needs to be transferred to the server while using AJAX based operations. The data has to be converted to JSON string before sending using JSON.stringify() method.

JavaScript Error Handling

Sometimes the JavaScript code may not run as smooth as expected due to several reasons like:

  • A problem with network connection
  • Invalid value in a form field
  • Referencing objects or functions that do not exist
  • Incorrect data being sent to or received from the server

These types of errors are known as runtime errors as they occur at the time the script runs.

The try...catch Statement

We can use the try...catch statement to trap the runtime errors, and handle them gracefully.

Any code that might raise an error should be placed in the try block of the statement and the code to handle the error is placed in the catch block.

Syntax:

try { // code that may cause an error } catch(error) { // action to be performed when an error occurs }

The try...catch...finally Statement

We can use the finally clause to execute, even if there is an error.

Syntax:

try { // code that may cause an error } catch(error) { // action to be performed when an error occurs } finally { // code executed regardless of error }

Throwing Errors

We can use the throw statement to throw an error manually.

Syntax: throw expression;

HTML5 Web Storage

HTML web storage provides two objects for storing data on the browser/client.

  • window.localStorage or localStorage
  • window.sessionStorage or sessionStorage

The localStorage Object

The localStorage object stores the data with no expiration date. The data will not be deleted until the data are deleted manually.

The data in localStorage object in the form of key:value pair.

Methods: The setItem method is used to set the items in localStorage while the removeItem method is used to remove the localStorage item.

The sessionStorage Object

The sessionStorage object acts the same as localStorage, except that the data are stored for only one session. When the browser/tab is closed the session will expire and data are lost.
